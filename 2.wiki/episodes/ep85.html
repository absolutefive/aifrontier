OpenClaw와 2026년 2월의 신호들

OpenClaw는 에이전트 하네스가 대중적 신호가 된 사건이고, 그 신호는 소프트웨어·SaaS·커리어의 도메인 갭을 암묵지와 타임 갭의 경쟁으로 압축한다.

핵심 포인트

  • OpenClaw는 Ralph Loop식 완전 자율 반복과 달리 Human In The Loop를 강조하면서도 강력한 도구 접근을 열어 시장에 큰 신호를 줬다.
  • Pi, VibeTunnel, Moltbook은 메신저와 에이전트 놀이터를 통해 자연어로 로컬 작업과 소프트웨어를 다루는 문턱을 낮췄다.
  • 샌드박싱 없는 접근은 강력하지만 위험하며, Moltbook의 API 키 유출은 OpenClaw가 예고한 보안 취약성을 빠르게 현실화했다.
  • Agent Teams와 shared task list는 에이전트 swarm이 조직 문화와 닮아가며 조향 가능성, 관측 가능성, 의도 명시가 중요해짐을 보여 준다.
  • SaaS 주가 폭락과 AI 로드맵 논의는 대중이 소프트웨어 자동화의 경제적 의미를 며칠 단위로 인식하기 시작했음을 보여 준다.
  • 남는 우위는 넓은 도메인 지식이 아니라 텍스트화·그래프화되지 않은 도메인 암묵지와, 그것을 빠르게 사업화하는 타임 갭이다.

다룬 흐름

신호

2월 들어 OpenClaw와 에이전트 도구가 쏟아지며 패러다임 변화의 시작을 감지한다.

하네스

Ralph Loop, OpenClaw, Pi, VibeTunnel, Moltbook의 철학 차이를 비교한다.

위험

샌드박싱 논쟁과 Moltbook API 키 유출로 강력한 에이전트의 보안 문제를 짚는다.

모델 전쟁

Claude Opus 4.6, GPT-5.3-Codex, Gemini 루머와 광고 전쟁을 통해 모델 경쟁의 가속을 본다.

에이전트 팀

Agent Teams, shared task list, C 컴파일러 실험으로 에이전트 swarm의 조직화 방식을 살핀다.

시장

SaaS 주가와 OpenClaw 대중화 사례를 통해 타임 갭이 며칠 단위로 줄어드는 현실을 논의한다.

암묵지

도메인 갭이 줄어든 뒤 남는 것은 context graph, ontology, graph RAG로 표현하기 어려운 암묵지라고 본다.

태도

빅테크 중력, 경제 양극화, 엔터테인먼트 경제, 호기심과 탐구의 태도로 마무리한다.

본문 엑기스

이번 회차는 2026년 2월 초, 쉬어가던 흐름이 다시 쏟아지기 시작했다는 감각에서 출발한다. 최승준은 작년 여름의 Ralph Loop가 PRD를 정하고 될 때까지 반복하는 문화였다면, OpenClaw는 다른 철학을 보여 줬다고 말한다. Peter Steinberger의 OpenClaw는 완전 자율 반복보다 Human In The Loop를 선호하면서도, 샌드박싱 없이 강력한 도구 접근을 열어 시장에 강한 신호를 줬다.

OpenClaw의 내부에는 Pi라는 에이전트 코어가 있고, Pi를 만든 Mario Zechner는 VibeTunnel처럼 Claude Code를 메신저에서 쓰는 흐름에도 관여했다. 이 계열의 도구들은 코드를 직접 다루기 어려운 사람도 디스코드나 메신저에서 자연어로 로컬 작업을 지시할 수 있게 문턱을 낮춘다. Pi의 철학은 MCP까지 많이 얹기보다 최소한의 직교적 기능을 두고, 에이전트가 필요한 소프트웨어를 스스로 만들어가게 하는 쪽에 가깝다.

Moltbook은 이 흐름의 또 다른 표면이다. 최승준은 Moltbook을 에이전트가 뭐든 할 수 있는 놀이터로 소개하며, 재미와 이야기 요소가 여전히 중요하다고 본다. '놀이는 인간과 기계의 삶'이라는 말처럼, 에이전트 도구는 단순 생산성 도구를 넘어 사람들이 만지고 놀아 보며 가능성을 체감하는 장이 된다. OpenClaw, Pi, Moltbook 모두 오래 개발해 온 40~50대 개발자들의 철학과 취향이 녹아 있다는 점도 중요하다.

하지만 강력한 도구 접근은 곧 보안 위험으로 이어진다. Pi를 만든 Mario는 샌드박싱을 진지하게 다루는 반면, OpenClaw는 훨씬 급진적이고 샌드박싱을 무시한 접근에 가까웠다. 그 위험은 몇 주도 지나지 않아 Moltbook의 API 키 대량 유출로 현실화됐다. 노정석은 유출도 100만 개가 되면 감흥이 덜해지는 이상한 시대라고 농담하지만, 최승준은 OpenClaw가 예고한 보안 취약성이 빠르게 현실이 됐다고 정리한다.

모델 전쟁도 같은 주에 겹친다. Claude Opus 4.6과 GPT-5.3-Codex가 거의 한 시간 차이로 발표되고, Gemini 3.5 루머도 돌았다. Anthropic은 광고 없는 Claude를 내세워 OpenAI의 광고 가능성을 비판했고, Sam Altman은 반응했다. 최승준은 AI 스트림에 광고가 들어갈 때의 상황을 보여 준 Anthropic 캠페인이 품질과 비즈니스 모델의 충돌을 드러낸다고 본다.

에이전트 swarm 쪽에서는 Agent Teams가 중요한 사례로 등장한다. 병렬 Claude 팀으로 Rust 기반 C 컴파일러를 만드는 실험은 단순히 sub-agent를 많이 띄우는 것이 아니라, shared task list와 작업 흔적, locking, 관측 가능성을 통해 팀처럼 일하게 만드는 방향을 보여 준다. 메인 에이전트가 모든 result를 받아 관리하던 방식에서, 에이전트 팀이 서로 task list를 공유하며 커뮤니케이션하는 구조로 이동하는 것이다.

이 변화는 인간 조직과 닮아 있다. 큰 규모의 에이전트 팀을 움직이려면 의도를 이끌어내고 명시하고 이해시키는 일이 중요해진다. 노정석은 이것이 하버드 비즈니스 리뷰에서 리더십을 말할 때 나오던 이야기와 같다고 본다. 최승준은 인간 사회에서 에이전트 사회를 배웠다면, 에이전트 사회에서 잘 작동하는 방식이 다시 인간 조직으로 피드백될 수 있다고 말한다.

OpenClaw가 대중적 인지도를 얻자 SaaS 주가가 크게 흔들린 일도 다뤄진다. 노정석은 테크 커뮤니티에서 이슈가 되고 대중에게 퍼지는 타임 갭이 이제 4~5일 단위로 줄었다고 본다. 예전에는 신정규가 말한 방향이 6개월 뒤 현실이 됐다면, 이제는 한 달 또는 며칠 만에 전 국민 투자자들이 OpenClaw와 SaaS 대체 가능성을 이야기한다. AI 변화가 시장 가격에 반영되는 속도 자체가 달라진 것이다.

AI 로드맵 논의는 더 큰 질문으로 이어진다. OpenAI가 2026년을 science의 해로 말하고, AI for Science와 RLVR 흐름이 모든 문제를 search 또는 learning problem으로 전환한다면, 소프트웨어에 가까운 일부터 빠르게 자동화된다. 노정석은 빅테크의 중력이 너무 강하기 때문에, 독립 플레이어가 탈출하려면 ChatGPT가 답할 수 없는 영역, 고객 데이터와 고객 문제, 도메인 지식이 있는 영역을 찾아야 한다고 말한다.

그런데 이 회차에서 중요한 전환은 도메인 갭이라는 표현 자체가 흔들린다는 점이다. 노정석은 대부분의 도메인에서 0부터 95까지는 프런티어 모델이 이미 잘하고, 남는 5%만 도메인 암묵지라고 본다. 철강, 화장품, 로펌처럼 결정적인 장인이 샤워하다 떠올리는 전략 같은 것이 남은 우위다. 이 암묵지는 텍스트로 모두 쓰거나, ontology, context graph, graph RAG 같은 관계 구조로 표현해야 모델 앞에 붙일 수 있다.

코딩 이후의 세계에서는 수혜자와 피해자가 갈린다. 가장 큰 수혜자는 엔지니어링 암묵지와 비즈니스 감각을 함께 가진 사람이고, 두 번째 수혜자는 소프트웨어는 몰라도 도메인 문제 의식과 evaluation metric을 가진 사람이다. 반대로 단순히 코딩 스킬을 우위로 삼던 많은 엔지니어는 AI slop을 많이 만들 수는 있지만 고객 개발 없이 제품만 양산하는 피해자가 될 수 있다. 그래서 문제 정의와 고객 개발이 제품 개발보다 더 중요해진다.

빅테크의 메시지는 이 흐름을 더 강하게 밀어붙인다. 최승준은 GPT-5.3-Codex 발표 뒤 OpenAI가 오래된 공업사나 가족 비즈니스를 modernizing하는 영상을 낸 것을 프로파간다처럼 느꼈다고 말한다. 노정석은 AI가 또 한 번 생산 수단의 효율을 끌어올리며 경제 구조를 squeezing할 수 있다고 보고, 단기적으로는 해고와 구조조정, 재정 정책, 기본소득 논의가 커질 수 있다고 본다.

새로운 직업군의 후보로는 재미와 엔터테인먼트 경제가 거론된다. 노정석은 초등학생들이 인플루언서와 연예인을 꿈꾸는 현상이 어쩌면 맞는 방향일 수 있다고 말한다. 생산성이 급격히 오른 뒤 남는 경제 활동은 사람을 끌어들이고 즐겁게 만드는 영역으로 이동할 수 있다. Elon Musk가 말하는 Universal Excessive Income 같은 SF적 상상도 더 이상 먼 농담처럼만 들리지 않는다.

마지막 결론은 확실한 해법보다 태도에 가깝다. 최승준은 표면적 기술을 빠르게 캐치업하는 것보다 만든 사람의 의도와 이야기를 음미하고, AI와 진지하게 마주하며 자기 경험을 쌓고 싶다고 말한다. 노정석은 Peter Diamandis의 Abundance를 떠올리며 한 사람이 여러 일을 벌릴 수 있다는 태도를 말하고, OpenClaw나 agent swarm이 그 가능성을 다시 보여 준다고 본다. 둘은 대처 방안은 없지만, 계속 읽고 말하고 생각을 끄집어내는 일이 울퉁불퉁한 미래를 건너는 방식이라고 마무리한다.
